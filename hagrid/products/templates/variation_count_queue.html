How it works

  • Thank you for counting remaining stock of some of our merch items.
  • You will be assigned an item to count.
  • Make sure you find all stock of that item. Leave full boxes closed and multiply by the expected amount.
  • Make sure you count the correct items. Check that product/fit/color/size are all matching.
  • If there are a lot (say, more than 100) of that item left, accurate counting is not that important. Try to be accurate to about 10%.
  • If there are only very few items left, try to get an accurate count.
  • Please count items in storage as well as items on display. Also check the try-on area.
  • If you cannot count something, click the respective button.
  • We try not to give two people the same item to count. Your item may be reassigned after about 5 minutes if you don't respond.
  • Please enter 0 if an item is out of stock.
  • You can enter your nickname for when we need to clarify stuff, but you don't have to. If you enter something, your browser remembers it for a while so you don't have to type again. We store it for the rest of the event and delete it from the database afterwards.
